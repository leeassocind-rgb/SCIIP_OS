# SCIIP Architecture

SCIIP is an asset-driven, event-sourced industrial intelligence operating system.

## Core Flow

Observation  
→ Command  
→ Event  
→ Graph Update  
→ Asset Profile Refresh  
→ Asset Timeline  
→ Dashboard

## Core Principle

ASSET is the permanent central object.

Everything connects to ASSET:
- Address
- City
- Zip
- Parcel
- Campus
- Property
- Alias
- Candidate
- Event
- Lease
- Sale
- Listing
- Owner
- Timeline
