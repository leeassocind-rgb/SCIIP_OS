Install:

```bash
chmod +x INSTALL.command
./INSTALL.command
```

The installer targets `~/Desktop/SCIIP_OS`, backs up every target file, records newly created files, patches only the Sprint 6 npm test script, runs Sprint 5 regression certification, and runs Sprint 6 certification.

```bash
chmod +x ROLLBACK.command

```
