# SCIIP Architecture

SCIIP_OS is the Southern California Industrial Intelligence Platform.

It is an asset-driven, event-sourced, knowledge-graph-native industrial operating system.

## Core Flow

Observation  
→ Command  
→ Event  
→ Knowledge Graph Update  
→ Asset Profile Refresh  
→ Asset Timeline  
→ Dashboard

## Core Principle

ASSET is the permanent central object.

Everything connects to ASSET.
