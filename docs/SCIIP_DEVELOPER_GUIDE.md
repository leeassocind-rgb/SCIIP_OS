# SCIIP Developer Guide

## Development Rules

- Git is the source of truth.
- Google Apps Script is the runtime target.
- Every module must strengthen the asset graph.
- Every processor must write commands.
- Every observation should become history.
- Do not duplicate helpers.
- Do not create duplicate normalize functions.
- Do not create duplicate business key helpers.
- Use SCIIP namespace prefixes.

## Namespace Prefixes

- sciipAddress
- sciipAsset
- sciipGraph
- sciipQueue
- sciipTimeline
- sciipProcessor
- sciipCommand
